## overview
Digital Electronics is one of the important courses in computer engineering classes at the university. We need to understand the fundamentals of computer architecture and use them to create our own architectures with hardware description languages like VHDL (or Verilog).

In this repository, we will work on some exercises that I got from [Dr. Marvasti's](https://khu.ac.ir/cv/331/english) class lectures.

Hope you enjoy it!

## Repository Structure

In the first part, you will find folders for each semester (for example, `042-051`). Inside each semester folder, there is a separate folder for each homework, named according to the homework number (e.g., `H1`).  

Inside each homework folder, you will find the following files:  
* **`homework_file.pdf`**: Contains the questions.  
* **`.jpg` files**: Architecture diagrams for questions that require architecture design.  
* **`.vhd` files**: VHDL code for the questions (including testbenches and implementations).  
* **`.qpf` file**: Project wizard file.


## How to Practice

For each homework, follow this suggested workflow:  

1. Carefully review the questions.  
2. Draw the architecture and write the corresponding VHDL code.  
3. Open the homework project wizard and verify your design with your code.  

Each question includes a VHDL file and a testbench file, which can be executed using ModelSim.
